## Short blurb

Spectrum is a desktop project cockpit for coding work: projects, workspaces, terminals, browser panels, file editing, and agent-driven workflows in one dense Electron app.

## Launch post

I am opening up Spectrum, the project cockpit I have been building for coding work.

It is an Electron desktop app that combines:

- project dashboards backed by SQLite
- multi-workspace canvases
- terminal, browser, file, and T3Code panels
- dense desktop-first navigation instead of a chat-only flow

The goal is simple: keep project context close to execution instead of scattering it across notes, terminals, browser tabs, and agent sessions.

Current status:

- early alpha
- macOS-first
- opinionated and still evolving quickly

If you want to try it, poke at it, or contribute, the repo is here:

- https://github.com/tomcrojo/spectrum

What I would especially like feedback on:

- workspace and panel UX
- project dashboard usefulness
- performance regressions
- Electron hardening and packaging
- where the app should stay opinionated vs become more configurable
